# steve-cooks/skills

Agent skills for AI-assisted development.

## Installation

```bash
npx skills add steve-cooks/skills
```

Or install a specific skill:

```bash
npx skills add steve-cooks/skills --skill whop-dev
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [whop-dev](skills/whop-dev) | Best practices for building Whop apps with Next.js, @whop/sdk, and Frosted UI |

## About

These skills extend AI coding agents (Claude Code, Cursor, GitHub Copilot, etc.) with specialized knowledge for building Whop-integrated applications.

## Contributing

Skills are located in the `skills/` directory. Each skill has its own folder with a `SKILL.md` file.
